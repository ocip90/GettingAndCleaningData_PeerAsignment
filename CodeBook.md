# CodeBook for Getting and Cleaning Data Peer Asignment (MOOC)

File result: tidyData.txt

# Procedure
1. Check if plyr is loaded
2. Check if zipped data is downloaded and unzipped
3. Instanciate raw data, and merged them
4. Subsetting data according with measurementes (mean, sd)
5. Naming activites
6. Labeling descripting variables
7. Create final tidy data (tidyData.txt)

# Identifiers
* subject: ID of test
* activity: Type of activity
